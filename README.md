# Restaurant Page
The Odin Project: Restaurant Page Project

## Project Summary
This project is a single-page restaurant website built as part of The Odin Project curriculum. It features a dynamic navigation system that loads different sections (Home, About, Menu) without full page reloads, creating a smooth user experience.

## Technologies Used
- **JavaScript**: Vanilla JavaScript for DOM manipulation and dynamic content loading
- **CSS**: Custom stylesheets for each page section
- **HTML**: Template structure for the main page
- **Webpack**: Module bundler for building and development
  - Webpack Dev Server for local development
  - Loaders for CSS and HTML processing
  - HTML Webpack Plugin for automatic HTML generation
